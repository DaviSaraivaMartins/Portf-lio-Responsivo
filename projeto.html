Nosso projeto de faculdade na disciplina de Design de Interface foi desenvolver um site para aluguel de casas, combinando funcionalidade e estética para proporcionar uma experiência de usuário atraente e intuitiva. Utilizamos HTML e CSS para criar uma interface responsiva e dinâmica, atendendo aos requisitos do projeto e às necessidades dos potenciais usuários.

HTML:

Utilizamos HTML (Hypertext Markup Language) para estruturar o conteúdo do site, garantindo uma organização lógica e acessível. Dividimos o site em seções distintas, como página inicial, listagem de casas disponíveis, detalhes individuais de cada propriedade, formulário de contato, entre outros. Incorporamos tags semânticas para melhorar a acessibilidade e a indexação pelos motores de busca, além de garantir uma base sólida para a estilização posterior com CSS.

CSS:

Com CSS (Cascading Style Sheets), aplicamos estilos visuais ao nosso site, tornando-o atraente e coeso. Utilizamos técnicas de layout responsivo para garantir que o site seja acessível em diferentes dispositivos e tamanhos de tela. Criamos uma paleta de cores harmoniosa, escolhemos tipografias adequadas e aplicamos efeitos visuais sutis para melhorar a experiência do usuário. Além disso, implementamos animações e transições para tornar a navegação mais fluida e envolvente.
